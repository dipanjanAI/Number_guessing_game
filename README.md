# Number Guessing Game 🎯

This is a beginner Python project where the user guesses a randomly generated number between 1 and 100.

## Features
- Random number generation
- Too high / too low hints
- Infinite guesses until correct
- Add attempt counter
- Add score system

## How to Run
1. Install Python
2. Download the file
3. Run:

python Guess_number_game.py
